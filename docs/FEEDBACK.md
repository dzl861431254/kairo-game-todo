## 🐛 发现的问题

### P1 - 体验问题（用户反馈）

1. **缺少主线剧情**
   - 游戏目前没有明确的主线目标
   - 玩家不知道最终要达成什么
   - 建议：添加主线任务/剧情线，如"重振门派"、"武林大会"等

2. **缺少场景切换**
   - 整个游戏停留在同一个界面
   - 缺乏视觉变化和沉浸感
   - 建议：添加不同场景（山门、练武场、江湖地图等）

### P2 - UI/交互问题

3. **按钮热区太小**
   - 招募/建造按钮需要非常精确的点击位置
   - 建议：增大按钮热区或添加点击反馈效果

4. **存档功能反馈不明确**
   - 点击存档按钮后没有明显的视觉提示
   - 建议：添加"已保存"toast提示

5. **候选弟子刷新机制不明**
   - 候选弟子列表在多月内保持不变
   - 建议：在UI中说明刷新条件
