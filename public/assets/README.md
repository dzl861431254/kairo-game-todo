# 江湖门派经营（MVP）美术/ UI 资产包（补齐版）

本包按照《美术素材准备任务说明（江湖门派经营·开罗味但不做像素）》的 **3.1 MVP必做清单** 补齐并统一打包。

## 1. 风格与规格（对齐文档）
- 风格：2D 轻Q版卡通（非像素），可爱但信息密度高
- 视角：俯视/轻俯视（约30°~45°），光源建议固定左上
- 文件：PNG（已做透明底）
- 摆放策略：推荐 **方案A 固定格子**（128×128 / 格）
  - 说明：本包是“源素材”，实际落地可按占格缩放到：
    - 1×1 → 128×128
    - 2×2 → 256×256
    - 3×2 → 384×256
    - 3×3 → 384×384

## 2. 目录结构与数量
- buildings/：20（10建筑 × L1/L2 = 20）
- props/：8（8）
- deco/：4（4）
- chars/：12（男01 6帧 + 女01 6帧 = 12）
- ui/icons/：18（18：资源6 + 标签8 + 状态4）
- vfx/：24（4类×6帧=24：build/upgrade/set_bonus/success）
- env/ground/：20（20）

合计 PNG：**106 张**（严格按文档 **3.1 清单逐项对齐**：20建筑 + 8设施 + 4装饰 + 12角色帧 + 18图标 + 24特效帧 + 20地表 = 106。文档中“110~120”是估算口径，通常会把一些额外变体/冗余帧也算进去）。

## 3. 命名规范
- 建筑：`building.<id>__L1__3x3.png` / `building.<id>__L2__3x3.png`
- 角色：`char.disciple.male01__idle_0.png` …
- UI：`icon.resource.silver.png`、`icon.label.qing.png`、`icon.status.wounded.png`
- 特效：`vfx.upgrade_0.png` …（按帧序号递增）

### A 建筑（房间）10×2
- 演武场/静室/藏经阁/药田/炼丹房/锻造房/医馆/客舍/议事堂/巡山岗哨（均含 L1/L2）

### B 设施/摆件 8×1
- 木人桩、香炉、书案、炉鼎、风箱、草药架、旗杆、木栅栏

### C 装饰 4×1
- 名匾、石狮、竹林景、影壁

### D 角色（弟子）2套
- 男01：idle2 + walk4
- 女01：idle2 + walk4

### E UI图标 18
- 资源6：银两/名望/传承/粮草/药材/铁料
- 标签8：清/烈/守/速/诡/邪/医/阵

### F 特效 24

### G 环境与地表 20
